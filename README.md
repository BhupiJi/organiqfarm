# OraganiqFarm.com Website

This is a simple static website for OraganiqFarm.com, focused on organic farming tips and monetization via affiliates and ads.

## Setup
- Hosted via GitHub Pages + Cloudflare.
- Edit HTML files to add content.
- Replace affiliate links and AdSense placeholders.

## Structure
- index.html: Home page
- about.html: About page
- contact.html: Contact form (static)
- blog.html: Blog list
- affiliates.html: Affiliate products
- blog-post-example.html: Sample blog post (duplicate for new posts)
- style.css: Styles
- script.js: Basic JS
- assets/: Images and favicon

Add more blog posts by copying blog-post-example.html and linking from blog.html.
